# Titan

[![npm version](https://badge.fury.io/js/%40titan%2Fcli.svg)](https://badge.fury.io/js/%40titan%2Fcli) [![Commitizen friendly](https://img.shields.io/badge/commitizen-friendly-brightgreen.svg)](http://commitizen.github.io/cz-cli/) [![Known Vulnerabilities](https://snyk.io/test/github/davidkelley/titan/badge.svg?targetFile=package.json)](https://snyk.io/test/github/davidkelley/titan?targetFile=package.json) [![semantic-release](https://img.shields.io/badge/%20%20%F0%9F%93%A6%F0%9F%9A%80-semantic--release-e10079.svg)](https://github.com/semantic-release/semantic-release) [![Greenkeeper badge](https://badges.greenkeeper.io/davidkelley/titan.svg)](https://greenkeeper.io/)

CLI providing developer workflow improvements for projects deploying AWS resources, like [CloudFormation](#), [CodeBuild](#) and [CodePipeline](#). _Introduces a flexible yet opinionated workflow, for deploying development/test branches and releasing change._

All functionality can be modified or changed through the use of plugins, which can be installed separately to the CLI.
